# Mock Data — x402-1CS Gateway

This folder contains realistic, x402-spec-compliant mock data for testing the
verifier and settler modules. The mocks use **real EIP-712 cryptographic
signatures** generated by ethers.js so they exercise the full verification path.

## Files

- `mock-wallets.ts` — Deterministic test wallets (facilitator + buyer)
- `mock-config.ts` — A complete GatewayConfig targeting Base mainnet USDC
- `mock-x402-payloads.ts` — Factory functions for EIP-3009 and Permit2 payloads
  with real EIP-712 signatures
- `mock-1cs-responses.ts` — Realistic 1CS API responses (quote, deposit, status)
- `mock-chain-reader.ts` — Configurable ChainReader stub for verifier tests
- `mock-settler-deps.ts` — BroadcastFn, DepositNotifyFn, StatusPollFn stubs
- `index.ts` — Re-exports everything for convenient imports
- `integration.test.ts` — End-to-end test: quote → verify → settle with all mocks
